## MobiDB 사용 방법
1. MobiDB.zip 압축을 풉니다.
2. 압축을 푼 폴더 안의 MobiDB.exe를 실행합니다.
3. 검색 범위를 선택합니다.
   - 1: 무기 / 방어구 / 엠블럼 룬
   - 2: 장신구 룬
4. 검색어를 입력합니다.


## DB 업데이트
프로그램 실행 시 GitHub의 최신 DB 버전을 확인합니다.
새 버전이 있으면 자동으로 data/mobidb.sqlite를 내려받아 교체합니다.

## 주의사항
MobiDB.exe만 따로 빼서 실행하지 말고, 압축을 푼 폴더 전체를 그대로 둔 상태에서 실행하세요.
_internal 폴더는 프로그램 실행에 필요한 파일입니다.
config 폴더에는 원격 DB 설정 파일이 들어 있습니다.
